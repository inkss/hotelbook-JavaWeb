## 酒店管理系统

> 项目具体说明：【Wiki】 JavaWeb 作业，即简单的酒店管理系统。

### 基础环境：

- Tomcat9
- Java 1.8+
- Mysql/Mariadb

### 描述：

很简陋的小项目，真的很简陋，毕竟是 17 年学生时代所写。

关于部署：Tomcat 务必为 9.X，JDK 可 12；数据库文件位于 `~/src/sql/` 中，初始登录用户密码为：`root`/`toor`；Tomcat 应用上下文为 `/hb`；项目依赖：`lib` 目录。
